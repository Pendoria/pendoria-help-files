Pendorian Elite UI

Created by Puls3 , Pendorian Elite UI is a userscript that started as a re-theme script for Pendoria.
As time passed, Pendorian Elite UI included many new features that changed the core of the UI.

It went through many versions, though by version 2.0 every feature was able to be toggled and customized.

Version History

Release v2.0

Version 2 of Pendorian Elite UI replaced the existing "if statement" code with objects containing data for each feature.

The final release of Pendorian Elite UI version 1 & 2 can still be viewed today
- Legacy Release v1.7.1
- Legacy Release v2.6

If you view the script content you can see major changes done to the structure.

Release v3.0

Version 3.0 of Pendorian Elite UI, completely replaced the entire core of Pendorian Elite UI's development process.

Instead of one large file written as one large file, it was split up into module files that were then imported into one single file for distribution. Due to the nature of other packing softwares, Puls3 had to write a completely brand new packer using NodeJS.

To the end user, version 3 is extremely similar to version 2. However, if you view the GitHub repository, the source files have been completely changed.
